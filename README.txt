ΓΡΑΠΤΗ ΑΝΑΦΟΡΑ ΠΡΟΓΡΑΜΜΑΤΟΣ ΜΕΡΟΣ-1
Στο terminal για να τρέξει η εργασία ακολουθούμε τα παρακάτω βήματα:
ΒΗΜΑ 1: cd (path για τον φάκελο-αρχείο)
ΒΗΜΑ 2: Εκτέλεση εντολής maven package για μεταγλώττιση προγράμματος
ΒΗΜΑ 3: Εκτέλεση της εντολής java -jar target/calendar-1.0-SNAPSHOT.jar [option] [filename] ή [filename]

Όταν δίνεται το [filename] μόνο σαν argument

  ΒΗΜΑ 1: Εμφάνιση menu και επιλογή χρήστη για το ποια λειτουργία θέλει να εκτελέσει (1. Προσθήκη νέου γεγονότος, 2. Αλλαγή κατάστασης μιας υπάρχουσας εργασίας, 3. Έξοδος προγράμματος-επανάληψης)

  ΒΗΜΑ 2: Στην περίπτωση που επιλέξει ο χρήστης την 1η λειτουργία (προσθήκη νέου γεγονότος) , χωρίζεται σε 2 είδη γεγονότων: σε appointments και tasks . 
          * Αν η επιλογή είναι appointments, θα χρειαστεί να δωθούν κάποια χαρακτηριστικά-δεδομένα: τίτλος, περιγραφή, εναρκτήρια μέρα, εναρκτήριος μήνας, εναρκτήριος χρόνος, εναρκτήρια ώρα, εναρκτήρια λεπτά.
            Έπειτα ο χρήστης θα χρειαστεί να κάνει μια ακόμα επιλογή σχετικά με το αν θα δώσει διάρκεια ή τελευταία μέρα, τελευταίος μήνας, τελευταίος χρόνος, τελευταία ώρα, τελευταία λεπτά.
            ( ο χρήστης θα δώσει μόνο το ένα απο τα 2 και με βάση αυτό θα δημιουργηθούν και τα αντίστοιχα δεδομένα του appointment)
            Αν δεν δωθεί διάρκεια , τότε δίνεται αυτόματα η τιμή -1, ενώ αν δωθούν: τελευταία μέρα, τελευταίος μήνας, τελευταίος χρόνος, τελευταία ώρα, τελευταία λεπτά τότε έχει δημιουργηθεί μέθοδος με την οποία προσαυξάνεται η διάρκεια με την εναρκτήρια ημερομηνία που αποτελεί το αποτέλεσμα του enddate.
          * Αν η επιλογή είναι tasks , θα χρειαστεί να δωθούν κάποια χαρακτηριστικά-δεδομένα: τίτλος, περιγραφή, τελευταία μέρα, τελευταίος μήνας, τελευταίος χρόνος, τελευταία ώρα, τελευταία λεπτά και για κατάσταση μπαίνει αυτόματα "IN-PROCESS", διότι μόλις δημιουργήθηκε το αντικείμενο.

  ΒΗΜΑ 3: Στην περίπτωση που επιλέξει ο χρήστης την 2η λειτουργία (αλλαγή κατάστασης μιας εργασίας), τότε θα εμφανιστούν όλες οι εργασίες. 
            Έπειτα ο χρήστης θα δώσει έναν τίτλο, τον οποίο το πρόγραμμα θα αναζητήσει στο αρχείο, και τότε θα δόθει η δυνατότητα στον χρήστη να τροποποιήσει την κατάσταση του αντίστοιχου αντικειμένου-εργασίας.

  ΒΗΜΑ 4: Στην περίπτωση που επιλέξει ο χρήστης την 3η λειτουργία, τότε τερματίζεται το πρόγραμμα, δηλαδή η επαναληπτική και η δυνατότητα επιλογής του χρήστη.

Όταν δίνεται το [option] και το [filename] σαν argument
    Το πρόγραμμα αρχικά κάνει sort τα γεγονότα από το πιο κοντινό χρονικά στην τωρινή ώρα μέχρι το πιο μακρινό χρησιμοποιώντας την μέθοδο calculateEpoch(μετράει τα δευτερόλεπτα που έχουν περάσει απο το 1 January 1970 00:00:00 μέχρι την ώρα που αρχίζουν τα Appointments ή την ώρα που έχουν προθεσμία τα Todos).Έπειτα τυπώνονται τα γεγονότα με την χρήση της μεθόδου showEventsFromNow.
    Στην συνέχεια αν ο χρήστης δώσει σαν option:
    1.day -> Εμφανίζονται τα γεγονότα απο την τωρινή ώρα μέχρι το τέλος της μέρας.
    2.pastday -> Εμφανίζονται τα γεγονότα από την αρχή της μέρας μέχρι την τωρινή ώρα.
    3.week -> Εμφανίζονται τα γεγονότα απο την τωρινή ώρα μέχρι το τέλος της εβδομάδας.
    4.pastweek -> Εμφανίζονται τα γεγονότα από την αρχή της εβδομάδας μέχρι την τωρινή ώρα.
    5.month -> Εμφανίζονται τα γεγονότα απο την τωρινή ώρα μέχρι το τέλος του μήνα.
    6.postmonth -> Εμφανίζονται τα γεγονότα από την αρχή του μήνα μέχρι την τωρινή ώρα.

ΓΡΑΠΤΗ ΑΝΑΦΟΡΑ ΠΡΟΓΡΑΜΜΑΤΟΣ ΜΕΡΟΣ-2

Στο terminal για να τρέξει η εργασία ακολουθούμε τα παρακάτω βήματα:

ΒΗΜΑ 1: cd (path για τον φάκελο-αρχείο)

ΒΗΜΑ 2: Εκτέλεση εντολής maven package για μεταγλώττιση προγράμματος

ΒΗΜΑ 3: Εκτέλεση της εντολής java -jar target/calendar-1.0-SNAPSHOT.jar

Για να κάνει ο χρήστης edit ένα γεγονός:

ΒΗΜΑ 4: Αφού τρέξουμε την εντολή , θα εμφανιστεί στην οθόνη ένα παράθυρο , στο οποίο ο χρήστης θα πρέπει αρχικά να πατήσει πάνω αριστερά το κουμπί «File» και έπειτα «Open», ώστε να φορτωθεί το αρχείο στο ημερολόγιο. Αν θέλει να επεξεργαστεί τα στοιχεία ενός γεγονότος θα πρέπει να επιλέξει και να πατήσει το κουμπί « Edit Events ».

ΒΗΜΑ 5: Αφού το πατήσει, θα του εμφανιστεί ένα ημερολόγιο όπου στο πάνω μέρος βρίσκονται τα κουμπιά «Previous» και «Next» όπου μέσω αυτών μπορεί να πάει στον προηγούμενο και στον επόμενο μήνα αντίστοιχα. Ανάμεσα από αυτά, υπάρχουν 2 Combo boxes μέσω των οποίων ο χρήστης μπορεί απευθείας να επιλέξει χρονιά και μήνα που θέλει να δει. Στο κάτω μέρος του παραθύρου υπάρχει το κουμπί «Back» όπου μπορεί να πατηθεί ανά πάσα στιγμή , ώστε να επιστρέψει στην αρχική σελίδα. Ο χρήστης πατάει πάνω στον αριθμό της ημέρας όπου θέλει να δει τα γεγονότα εκείνης της ημέρας.

ΒΗΜΑ 6: Στην δεξιά μεριά του παραθύρου θα εμφανιστούν ως κουμπιά τα γεγονότα (appointments και tasks ) εκείνης της ημερομηνίας , εφόσον υπάρχουν. Αν δεν υπάρχει κάποιο τότε θα εμφανιστεί το αντίστοιχο μήνυμα.

ΒΗΜΑ 7: Για να δει και να επεξεργαστεί τα στοιχεία ενός γεγονότος της συγκεκριμένης ημερομηνίας που έχει επιλέξει , θα πρέπει να πατήσει το κουμπί με το αντίστοιχο όνομα του γεγονότος.

Αν το γεγονός είναι appointment:

ΒΗΜΑ 8:  τότε αφού το πατήσει, θα του εμφανίζεται ένα μικρό παράθυρο όπου θα  ζητάει από τον χρήστη να επιλέξει τι θέλει να επεξεργαστεί, duration ή end-date. Εφόσον επιλέξει, θα κλείσει το μικρό παράθυρο και θα εμφανιστούν τα αντίστοιχα στοιχεία τα οποία επέλεξε να δει (duration ή end-date) μαζί με τον τίτλο , την περιγραφή, start day, start month, start year, start hours και start minutes.

Αν το γεγονός είναι task:

BHMA 8: τότε αφού το πατήσει, θα εμφανιστεί ο τίτλος, η περιγραφή, end day, end month, end year, end hours, end minutes και το status.

Όλα τα στοιχεία εμφανίζονται σε μορφή Combo box , εκτός από τον τίτλο και την περιγραφή που εμφανίζονται σε Textfield. Τα κουτάκια εμφανίζονται συμπληρωμένα ήδη με τα αποθηκευμένα στοιχεία τους.

ΒΗΜΑ 9:  Αφού αλλάξει ο χρήστης τα στοιχεία που επιθυμεί, πατάει το κουμπί «Save», ώστε να αποθηκευτούν οι αλλαγές στο αρχείο.

Για να προσθέσει ο χρήστης ένα γεγονός: 

1ο ΒΗΜΑ:Εφόσον ανοίξετε την εφαρμογή κάνοντας βήμα-βήμα τις οδηγίες εκτέλεσης,τότε θα χρειαστεί να επιλέξετε το αρχείο που θα θέλατε να ανοίξει σαν ημερολόγιο ,πατώντας file και μετά open.

2ο ΒΗΜΑ:Πατάτε την επιλογή add Events.

3o ΒΗΜΑ:Επιλέγετε την ημερομηνία που θέλετε στο Calendar,και μετα χρειάζεται να απαντήσετε εάν θέλετε να προσθέσετε Appointment event ή Task event.

4ο ΒΗΜΑ:Εάν η προηγούμενη επιλογή ήταν Task event τότε απλα στο επόμενο παράθυρο θα πρέπει να συμπληρωθούν τα υπόλοιπα στοιχεία και να πατηθεί το κουμπί  save ώστε να αποθηκευτεί το νέο γεγονός.Ενώ ,άμα η προηγούμενη επιλογή ήταν Appointment event τοτε θα έρθετε πάλι σε θέση επιλογής για το αν θέλετε να κάνετε προσθήκη με βάση το end-date ή duration,και στις δύο επιλογές στο επόμενο παράθυρο θα χρειαστεί να προσθέσετε και τα υπόλοιπα στοιχεία και να πατήσετε και το κουμπί save ώστε να αποθηκευτεί το νέο γεγονός.
